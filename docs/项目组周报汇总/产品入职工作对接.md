# 一、周报

1.  每周一下午五点固定周会

    1.  首次进入时：提前调试声音与共享屏幕。

    2.  提前准备：根据团队提供的个人总结，整理周报至<a href="https://hertzflow.atlassian.net/wiki/spaces/H/pages/10518529" data-linked-resource-id="10518529" data-linked-resource-version="71" data-linked-resource-type="page">项目组周报汇总</a>

    3.  团队每周五下班前更新周报至前后端整合群置顶folder`周报`。

2.  周报给ai总结即可。prompt参考见<a href="https://hertzflow.atlassian.net/wiki/spaces/H/pages/edit-v2/82182190#%E5%91%A8%E6%8A%A5%E7%94%9F%E6%88%90-prompt" rel="nofollow">附录</a>

# 二、月报

1.  每月最后一周周三需给出当月月报，英文版。流程为：**月底那周周三交付：**<a href="https://hertzflow.atlassian.net/wiki/spaces/H/pages/10518529" data-linked-resource-id="10518529" data-linked-resource-version="71" data-linked-resource-type="page"><span style="background-color: rgb(211,241,167);">周报</span></a>喂给claude -\> 整体过一下，分段调整叙述 -\> 补充技术细节与分工与甘特图 -\> 整理到设计给的模版上，导出pdf发到产品群 @arthur确认

2.  周报给ai总结即可，建议文字部分备份到月报文档上。prompt见附录。

# 四、Vibe PMing

1.  前置：购买Claude Code Max，下载并安装<a href="https://code.claude.com/docs/en/vs-code#install-the-extension" class="external-link" rel="nofollow">Visual Sudio Code</a>

    1.  接收SMS：可查找SMS接收器或淘宝购买支持国家的电话卡（比如Vodafone）

    2.  Claude 不支持部分亚洲地区，尽量VPN链接欧美地区

    3.  点击VPN 的Copy Shell Command，打开电脑终端Terminal并复制即可绕开大部分监管。这里找不到可询问ai “`我用的代理软件是XXX，在哪里能找到HTTP port，SOCKS port，用于直接复制到电脑终端，在 shell 里设置代理环境变量`”

2.  <a href="https://ccforpms.com/getting-started/installation" class="external-link" rel="nofollow">VS Code中安装Claude Code Max</a>

    1.  终端中安装Claude Code 复制粘贴代码

        curl -fsSL https://claude.ai/install.sh | bash

        </div>

# 五、发版流程

环境推进顺序：Testnet → Preview → Production

<div class="code panel pdl" style="border-width: 1px;">

<div class="codeContent panelContent pdl">

``` syntaxhighlighter-pre
合约开发
├─ 需求确认
├─ 代码修改
├─ 安全验证
└─ Testnet部署
└─ 回归测试，产品验收 （功能 + 计算 + 冒烟测试 + 压测）
├─ Keeper & 后端跟进，适配，Preview部署
├─ 前端跟进适配，迁移，Preview部署

生产发版
├─ 合约部署
├─ Keeper & 后端接口与依赖确认，部署
├─ 前端部署 + 域名配置（如有新子页面）
├─ 测试，产品确认推生产
└─  后续跟进（盯峰值，用户反馈等）
```
